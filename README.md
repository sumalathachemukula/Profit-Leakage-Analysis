# 📊 Profit Leakage Analysis

## Project Overview
This project aims to identify key factors contributing to profit loss (leakage). By analyzing sales, discounts, regions, and product categories, this Power BI dashboard helps businesses understand where profits are being lost and how to improve overall profitability.

## Repository Contents
* **PROFIT LEAKAGE.pbix**: Interactive Power BI dashboard with visualizations and DAX measures.
* **superstore.csv**: Dataset used for the analysis.
* **Dashboard Preview**: Screenshot of the final dashboard.

## Key Analysis Areas
* **Profit Leakage:** Identification of loss-making areas.
* **Discount Impact:** Understanding how high discounts affect profit.
* **Regional Analysis:** Comparing profit and loss across regions.
* **Product Categories:** Finding categories contributing to losses.
* **Customer Insights:** Identifying top customers causing losses.
